# Pakar Pinus - Android Expert System

Pakar Pinus is an Android-based expert system application designed to diagnose pests and diseases in pine trees based on observable symptoms. This application implements the Forward Chaining inference method combined with Certainty Factor to determine the most probable disease affecting the plant.

## Features
- Symptom-based disease diagnosis
- Expert system using Forward Chaining
- Certainty Factor calculation for diagnosis confidence
- Disease information and treatment recommendations
- Diagnosis history storage
- User-friendly mobile interface

## Technologies Used
- Java
- Android Studio
- Firebase Firestore
- Glide (for image loading)
- Artificial Intelligent

## How It Works
1. Users select symptoms observed on the pine tree.
2. The system processes the symptoms using the Forward Chaining method.
3. Certainty Factor is calculated to determine the probability of each disease.
4. The system displays the most probable disease along with its treatment recommendation.
